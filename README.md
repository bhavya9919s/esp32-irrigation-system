# esp32-irrigation-system
Edge-AI irrigation using ESP32

## Overview
This project implements a smart irrigation system using ESP32-S3 and a soil moisture sensor to automate water supply based on real-time soil conditions.

## Features
- Real-time soil moisture monitoring  
- Automatic water pump control  
- Edge-based decision making (no cloud required)  
- Low-cost and energy-efficient system  

## Hardware Components
- ESP32-S3  
- Soil Moisture Sensor  
- Relay Module  
- Water Pump  

## Working Principle
The soil moisture sensor collects analog data, which is processed by the ESP32-S3. Based on a predefined threshold, the system automatically turns the water pump ON or OFF.
